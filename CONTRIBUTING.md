# Contributing / Permission Requests

Thank you for checking out this portfolio repository. This project is a personal portfolio and is not open-source for reuse.

If you would like permission to reuse or fork code from this repository, please do one of the following:

- Open an issue requesting permission and include how you plan to use the code.
- Contact me via the contact information on my GitHub profile (https://github.com/MoodyJW).

If permission is granted, I will respond with explicit written permission and (where appropriate) add a permissive license or provide a separate code excerpt with licensing terms.

Do not assume permission: copying or republishing code from this repository without explicit consent is not allowed.
